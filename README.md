# Auto-tagging-customer-support
Auto-Tagging Customer Support Tickets Using LLM
Project Overview

This project automatically classifies and tags customer support tickets using a Large Language Model (LLM). The system analyzes ticket descriptions and predicts the most relevant categories, helping support teams organize, prioritize, and route tickets more efficiently.

Features
Automatic ticket categorization
Zero-shot classification using pre-trained LLMs
Displays top predicted tags with confidence scores
Reduces manual ticket sorting effort
Easy deployment with Python and Hugging Face Transformers
Technologies Used
Python
Pandas
Hugging Face Transformers
BART Large MNLI (Zero-Shot Classification)
Google Colab
Workflow
Load customer support ticket data.
Define candidate ticket categories.
Use a zero-shot classification model to predict tags.
Display the top matching categories with confidence scores.
Analyze and evaluate tagging performance.
Example Output

2. Account Management (0.05)
